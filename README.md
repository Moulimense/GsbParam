Clonage du projet Récupère le dépôt sur ton ordinateur avec git clone .

Création d'une branche Ne travaille jamais sur la branche principale. Crée ta propre branche avec git checkout -b nom-de-ta-tâche.

Développement et sauvegarde Fais tes modifs, ajoute-les avec git add . et valide-les avec git commit -m "Description de ce que tu as fait".

Envoi sur GitHub Envoie ton travail sur le serveur avec git push origin nom-de-ta-branche.

Ouverture d'une Pull Request (PR) Sur l'interface GitHub, clique sur "New Pull Request" pour proposer tes changements à l'équipe.

Fusion (Merge) Après vérification par les autres, fusionne ta branche dans le projet principal (main).

Mise à jour Avant de recommencer, reviens sur main et tape git pull origin main pour récupérer le travail des autres.
